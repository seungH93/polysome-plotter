---
name: Multi-Sample Comparison
description: Overlay up to 5 polysome samples (Control + Cases) with per-sample X/Y shift sliders, tube dividers, and Fraction/Time toggle.
type: feature
---
- Up to 5 samples overlaid on one chart. First upload = Control (black), subsequent = Cases (red, blue, green, purple).
- Each sample has independent xShift / yShift sliders for manual alignment to Control.
- Tube dividers: evenly split full X range into N tubes (user-configurable, default 26), shown as dashed vertical lines with T1..TN labels.
- X-axis toggle: Fraction number ↔ Time (min). Time mode requires user to enter total minutes; mapping is linear across X range.
- Sample list shows color swatch, editable name, visibility toggle, reset alignment, remove buttons.
